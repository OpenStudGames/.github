### OpenStud

![A blocky, old Roblox style logo, that has the text OpenStud](https://github.com/user-attachments/assets/39f8dd8f-f9ea-4d3d-95da-6181b496d1d1)

OpenStud is a fun open-source Roblox clone licenced under the GNU AGPL. The name comes from the unit of measurement used in Roblox games.
